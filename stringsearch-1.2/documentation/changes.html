  • Release 1.1 (09.03.2004): Bugfix release. Bugs were found in the preprocessing codes for bytes. This release fixes these bugs.
  • Release 1 (Feb 2004, Build 100 200402172142 ): First public release. Contains
    • BNDM
    • BNDM Wildcards
    • Boyer, Moore, Horspool
    • Boyer, Moore, Horspool, Raita
    • Boyer, Moore, Sunday
    • Shift-Or
    • Shift-Or Classes
    • Shift-Or Mismatches
    • Shift-Or Wildcards
