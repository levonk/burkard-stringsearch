StringSearch - high-performance pattern matching algorithms in Java - documentation

Table of contents - 1. Introduction

The Java language lacks fast string searching algorithms. The StringSearch library provides implementations of algorithms of the Boyer-Moore family and the Shift-Or (bit-parallel) family. These algorithms are easily five to ten times faster than the naive implementation found in java.lang.String.

This library contains implementations of the following pattern matching algorithms:

  • General purpose algorithms:
    • BNDM (algorithm invented by Gonzalo Navarro and Mathieu Raffinot)
    • Boyer, Moore, Horspool
    • Boyer, Moore, Horspool, Raita
    • Boyer, Moore, Sunday
    • Shift-Or (algorithm invented by Ricardo Baeza-Yates and Gaston Gonnet)
  • Searching with wildcards (don't-care-symbols):
    • BNDM Wildcards
    • Shift-Or Wildcards
  • Searching with character classes:
    • Shift-Or Classes
  • Searching with mismatches:
    • Shift-Or Mismatches
